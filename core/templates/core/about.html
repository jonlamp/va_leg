About VAL

Virginia Legislation (VAL) is a website dedicated to tracking and displaying information about the bills being discussed in Virginia's Legislature. This site maintains separate copies of the bills hosted by the Legislative Information System (LIS), and allows users to search for bills through a variety of ways. VAL is a new website, so new features are constantly being added! Currently in the works are user accounts that will allow you to follow bills that you care about, make notes about them, and share your notes with others. Be sure to check back for updates!

This website would not be possible without the dedication of the Division of Legislative Automated Systems, who maintains LIS and provides easily accessible copies of bill and legislator information through helpful CSV files. These files are accessed once each night, so information on VAL is only ever as up to date as the records available from the previous night. To find out more information about these files, or how VAL collects and parses these files, check out the source code for this project on GitHub.
